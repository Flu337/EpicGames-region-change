# VPN Gate - Инструкция по установке и использованию

## Для зуба
Быстрая настройка VPN для изменения региона в играх и приложениях.

## Установка

### 1. Скачай архив
Скачай последнюю версию VPN Gate Client с официального сайта или из этого репозитория.

### 2. Достань из ZIP
Распакуй архив в удобную папку на компьютере.

### 3. Запусти файл
Запусти `vpngate-client-v4.44-9807-rtm-2025.04.16.exe`

## Использование

### 1. Зайди в приложение
После запуска откроется окно SoftEther VPN Client Manager

### 2. Перейди в "VPN Gate Public VPN Relay Servers"
В главном окне программы выберите эту опцию.

### 3. Выбери страну
В списке доступных серверов:
- Найди нужную страну
- Выбери сервер с хорошей скоростью (обрати внимание на пинг)
- Нажми на нее

 

## Для Epic Games и других сервисов
После подключения к VPN:
1. Запусти лаунчер Epic Games
2. зайди в профиль
3. и меняй страну
